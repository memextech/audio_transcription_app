## Requirements

- macOS 11.0 or later
- Apple Silicon Mac (M1/M2/M3)
- Python 3.11+
- Microphone permissions

## Quick Start

1. Clone the repository:
```bash
git clone https://github.com/memextech/audio_transcription_app.git
cd audio_transcription_app
```

2. Run setup script:
```bash
chmod +x setup.sh
./setup.sh
```

3. Launch the app:
```bash
open "Audio Transcriber.app"
```
